Data Binding


El Data Binding en AngularJS es la sincronización atuomática
de datos entre el modelo y la vista. La vista siempre muestra los datos que contiene el modelo.
Quiere decir que cuando el modelo cambia, la vista se actualiza.
Pero también la vista puede cambiar y AngularJS hará que el modelo
también sea actualizado.
